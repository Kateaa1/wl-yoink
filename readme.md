# wl-yoink

wl-yoink is a simple command line tool that copies the contents of a file into your wl-clipboard

## dependencies 
wl-copy

## build deps
make
gcc

## build instructions
`git clone https://github.com/Kateaa1/wl-yoink && cd wl-yoink && make install`

## Usage
`wl-yoink [target file] # Copies file into clipboard`
`wl-yoink --help`

## Example
`wl-yoink src/main.c`
